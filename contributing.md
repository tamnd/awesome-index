# Contributing

This index is auto-generated from the [sindresorhus/awesome](https://github.com/sindresorhus/awesome) source list. The data is refreshed daily via GitHub Actions.

## Suggesting changes to the source list

If you want to add, remove, or update an entry in the awesome list itself, please visit [sindresorhus/awesome](https://github.com/sindresorhus/awesome) and open a pull request there.

## Improving the generator

If you want to improve how this index is generated, rendered, or structured:

1. Fork this repo.
2. Make your changes in `src/awesome_index/`.
3. Run `uv run awesome-index` locally to verify the output.
4. Open a pull request.

## Reporting issues

If you notice incorrect data, broken links, or formatting problems, please [open an issue](https://github.com/tamnd/awesome-index/issues/new).
